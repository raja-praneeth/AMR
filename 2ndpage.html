Web AR: Augmented Reality

What is Web AR?

Web AR (Web-based Augmented Reality) is a cutting-edge technology that allows users to experience augmented reality directly in a web browser — without downloading any apps. It combines camera input, 3D models, and spatial tracking to overlay digital content into the real-world environment.

Why Web AR?

Traditional AR experiences often require native apps, which limit accessibility. Web AR makes AR experiences available instantly across devices, reaching a wider audience.

  • No app downloads or installations
  • Cross-platform (Android, iOS, desktop)
  • Instant load and share via URL
  • Ideal for education, retail, product demos, and architecture

Our Project

We built a full Web AR experience where users can place 3D objects in their real-world space using just a browser. Our system overlays these objects using the device’s camera and lets users interact with them in real-time.

  • Live camera feed with overlay
  • 3D models rendered interactively
  • Touch and drag to reposition objects

Technologies Used

  • HTML/CSS/JS Frontend, styling, and behavior
  • <model-viewer> 3D model visualization
  • WebXR / getUserMedia Camera and AR APIs
  • JavaScript logic for search, login, and interface control
